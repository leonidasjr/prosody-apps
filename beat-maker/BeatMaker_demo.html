How to use in a pronunciation class

Phase 1 — Perceive
Listen + mark stress
Play L1_Beat_and_Voice twice. Learners mark stressed syllables on the transcript while tapping along with the beat pulses.
Phase 2 — Compare
Project the plot
Project BeatMaker_plot.jpg. Discuss where F0 peaks differ between the red (L1) and blue (L2) contours and why.
Phase 3 — Produce
Record → re-run
Learner records the utterance → save as L2_name.wav → re-run BeatMaker → project the new plot for immediate visual feedback.
Research note
Proven for intonation
Pre/post-test with 10 Brazilian EFL learners showed significant intonational convergence toward native patterns after BeatMaker training (Silva Jr., 2023).
